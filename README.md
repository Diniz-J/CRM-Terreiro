# Teiunecc Admin

Sistema de gerenciamento para Terreiro de Candomblé/Umbanda.

## Sobre

Sistema modular desenvolvido em Go para gestão completa de terreiro:
- Cadastro e gestão de membros
- Controle de mensalidades
- Registro de presença via QR Code

## Status

Em desenvolvimento - FASE 1: CRUD de Membros

## Tecnologias

- Backend: Go 1.21+
- Banco de Dados: MySQL 8.0
- Containerização: Docker
